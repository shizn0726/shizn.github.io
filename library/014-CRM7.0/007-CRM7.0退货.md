# CRM7.0退货
在CRM7门店管理模块，点击![crm7](picture\\CRM7\\43.png)按钮，进入CRM7退货页面。  
![crm7](picture\\CRM7\\44.png=500-)  
![crm7](picture\\CRM7\\45.png=500-)  
## 选择商户
显示停用的集团信息、门店信息、原终端数和终端数。  
集团信息包括集团图片，集团名称和集团编号。  
门店信息包括门店图片，门店名称和门店编号。  
原终端数显示框默认置灰，不可修改，数字默认为原终端数。   
## 产品配置
用户进行退货操作时候不可对产品配置版本信息进行修改，选中状态为![crm7](picture\\CRM7\\46.png)，其余为未选中。  
开通时间格式为年-月-日。  
## OA编号
可在输入框输入OA编号，非必填。  
## 应付款总计
应付款总计默认为0，可在备注框输入备注信息，非必填。  
完成商户内容后点击![crm7](picture\\CRM7\\48.png)按钮完成加终端申请操作。
